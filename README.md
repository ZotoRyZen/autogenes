### PDF Links

To add actual PDF files, update the `onclick` handlers in the Codex section:

```javascript
// Change from:
onclick="window.open('#','_blank')"

// To:
onclick="window.open('pdfs/your-file.pdf','_blank')"
```

### Colors

Modify CSS custom properties in the `:root` selector:

```css
:root {
    --bg: #050505;
    --amber: #ffb800;
    --green: #00ff41;
    --red: #ff0040;
    --cyan: #00ffff;
}
```
